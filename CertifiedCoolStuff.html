Making a drop down selection menu for links or other content!

Need something to drop down when necessary? Use this technique!

Requirements: a parent and child element. Example below:

Look at me mom!
I see you kiddo!
hello
hello

<div id="upper">
    <a>insert text</a>
    <div id="lower">
        <a>insert text</a>
    </div>
</div>

With the following css to make the bottom element appear after everything:

#upper{
    width:200px;
    color:/any/;
}
#lower{
    display:none;
    color:/not upper's/;
    position:absolute;
}
#upper:hover #lower{
    display:block; }

#upper and #lower can be substituted for any class or id, as long as there are nested divs.
